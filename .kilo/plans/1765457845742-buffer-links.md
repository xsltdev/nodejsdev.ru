# План: Замена реф-ссылок на md-линии в buffer.md

## Цель

Заменить внутренние референсные ссылки (например, `[require()][require()]`) на стандартные Markdown-ссылки, используя определения из блока в конце файла, чтобы обеспечить соответствие стилю текущей документации Node.js.

## Анализ текущих ситуаций

Найдено 4 случая реф-ссылок:

1. `[UTF-8][utf-8]` (строка 161) — внешняя ссылка, нужен anchor
2. `[RFC 4648, раздел 5][rfc 4648, section 5]` (строка 173) — внешняя ссылка, нужен anchor
3. `[ASCII][ascii]` (строка 179) — внешняя ссылка, нужен anchor
4. `[Web Streams API][web streams api]` (строка 467) — внешняя ссылка, нужен anchor

## Шаги реализации

1. Найти текущий файл: `docs/api/buffer.md`
2. Проверить, не используется ли в файле уже определённых ссылок:
    - `[utf-8]` → `utf-8`
    - `[ascii]` → `ascii`
    - `[base64url]` → `base64url`
    - `[web streams api]` → `web streams api`
3. Заменить реф-ссылки на внешние anchor-ссылки:
    - `[utf-8]` → `utf-8`
    - `[ascii]` → `ascii`
    - `[base64url]` → `base64url`
    - `[web streams api]` → `web streams api`
4. Провести финальную проверку:
    - Убедиться, что все внешние ссылки работают
    - Проверить отсутствие синтаксических ошибок

## Примеры замены

```diff
[UTF-8][utf-8]` → `utf-8`
[ASCII][ascii]` → `ascii`
[base64url][base64url]` → `base64url`
[web streams api][web streams api]` → `web streams api`
```
