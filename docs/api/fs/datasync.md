# datasync

## FileHandle

```js
filehandle.datasync();
```

???quote "История"

    Добавлено в: v10.0.0

**Возвращает:**

-   `<Promise>` &mdash; Выполняется с `undefined` в случае успеха.

Принудительно переводит все операции ввода-вывода в очереди, связанные с файлом, в состояние завершения синхронизированного ввода-вывода операционной системы. За подробностями обратитесь к документации POSIX [`fdatasync(2)`](http://man7.org/linux/man-pages/man2/fdatasync.2.html).

В отличие от [`filehandle.sync`](sync.md#filehandle) этот метод не удаляет измененные метаданные.
